Règlement

Chapitre I : Règles de base.


1.1 - Le personnel de police se doit de respecter sa hiérarchie.
1.2 - Il est bon de vivre en collectivité avec ses collègues de travail.
1.3 - L’arme de service ne peut en aucun cas être gardée en dehors de ses heures de service.
(n’a pas lieu jusqu’à nouvelle ordre) 1.4 - Votre contrat d’embauche est de 7 Jours minimum sans possibilité de démissionner avant ce délai.
1.5 - L’agent des forces de l’ordre se doit d’avoir une conduite exemplaire.
1.6 - Un agent en service est obligé de servir et défendre ces habitants.
1.7 - Vous n’avez nul droit de refuser un ordre d’un supérieur (sauf abus de pouvoir justifié)
1.9 - L’utilisation de la radio est utilisable uniquement lorsque l’agent est en service.les communication radio sont prioritaire,il est formellement recommandé de ne pas s'en servir pour des message personnel.L’abus peut être sanctionner.

1.1.1 - Tout conflits entre agent de police doit être expliqué à un supérieur.
1.1.2 - L’uniforme doit rester au sein des locaux lorsque vous quitter votre service.
1.1.3 - Les véhicules de police ne doivent pas servir pour des raisons personnelles.
1.1.4 - Le casier judiciaire d’un agent de police doit être vierge.

Chapitre II : Les Contrôles.


2.1 - Un agent ne peut arrêter une personne sans raison particulière.
2.2 - Un agent ne peut mettre d’amende plus élevé que le code pénal.
2.3 - Lors d’un contrôle on peut vous demander votre badge.
2.4 - Lors d’une arrestation, l’agent se doit de prononcer le serment et les droits de l’individu.
2.5 - Un agent n’étant pas en service ne peut interpeller un individu.
2.6 - Un agent ne peut en aucun cas arrêter un individu parce que sa tête ne lui revient pas.
2.7 - Lors d’une interpellation sur la route, un agent ne peut bloquer ou gêner la circulation.

Chapitre III : L’extérieur.


3.1 - Un agent ne peut patrouiller seul, il faut être minimum deux.
3.2 - Un agent ne peut encombrer les routes sans raison.
3.3 - Un agent se doit lors de mise de barrière, mettre une déviation pour la circulation.
3.4 - Autorisation de tirer dans les pneus lors d'un délit de fuite avec conduite extrêmement dangereuse uniquement (et power gaming: sens inverse sur l'autoroute, stunt, roule sur le trottoir etc) pour les légers délits de fuite genre le mec roule juste vite sur la route proprement essayer de l'arrêter à la loyale. le gars de toute façon n'aura pas d'essence de façon illimité.
3.5 - Un agent peut appeler des renforts uniquement si l’intervention en requiert.
3.6 - Lors d’un braquage ou prise d’otage aucun agent ne peut intervenir seule, laisser faire les négociateurs.(obligation de brancher sa Gopros pour éviter tout dérapage et action indésirable)
3.7 - Un agent se doit d’appeler une dépanneuse lorsqu’un véhicule est saisi.
3.8 - Un agent ne peut s’introduire dans une propriété privée sans mandat d’un supérieur.
3.9 – Un agent ne peut prendre un autre véhicule de police autre que le siens desservie.

Chapitre VI : Motif de licenciement.


4.1 – Trafic d’arme des services de police
4.2 – Vendre de la drogue saisie.
4.3 – Utilisé son véhicule pour des fins personnelle.
4.4 – Utiliser ton arme hors service.
4.5 – Arrêté quelqu’un pour un délit de faciès.
4.6 – Association avec des malfaiteurs.
4.7 – Faire grève pour une raison in valable.
4.8 – Ne pas travailler plus de 5 jours consécutifs.
